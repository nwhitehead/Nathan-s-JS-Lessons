Because objects are values, and properties of objects have a name and a value, you can nest objects inside of objects. Here's a scientific apple object:

To access properties of objects, use the notation object.property. So scientific_apple.owner evaluates to "Nathan". If the property has a value that is an object, you can use the dot notation again to dig deeper. So scientific_apple.classification.cultivar evaluates to "Gravenstein".
